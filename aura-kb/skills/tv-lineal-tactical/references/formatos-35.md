# Catálogo de 35 Formatos Comerciales — TV Lineal

## PTR — Publicidad Tradicional

| # | Formato | Duración | KPI principal | Contexto óptimo |
|---|---------|----------|---------------|-----------------|
| 01 | **Spot comercial** | 5"-120" (std 20") | GRP / TRP / CPR | Base de cualquier campaña |
| 02 | **Spoteo** | 5"-10" | Frecuencia / GRP | Eventos de alta audiencia, prime time |
| 03 | **ELE / Cascada** | 2-4 spots consecutivos | Recuerdo espontáneo | Lanzamientos, narrativa encadenada |
| 04 | **Cut-in** | Mín. 20" | Atención / notoriedad | Deportes, realities — antes del resultado clave |
| 05 | **Infomercial** | 30 min | Leads / ventas | Madrugada, respuesta directa |
| 06 | **Televenta / Teletienda** | Canal o franja | Ventas directas | Canal propio, DRTV sostenido |

| 09 | **Telepromoción (TP)** | 45"-120" | Consideración | Matutinos, magazines — el conductor avala |

| 14 | **Mención con demostración** | 30"-90" | Intención de compra | El conductor usa el producto en cámara |

| 16 | **Patrocinio de programa** | Temporada | Brand equity | Asociación profunda con los valores del programa |
| 17 | **Patrocinio apertura/cierre (careta)** | 5"-15" | Notoriedad premium | Posición privilegiada, máxima atención |

## Regla de selección de formato

Para cada objetivo, cruza tres variables:
- **Etapa del funnel** (awareness / consideración / conversión / lealtad)
- **Nivel de disrupción aceptable** (alta = PTR; baja = PNT/gráficos)
- **Profundidad del mensaje** (micro = cortinilla/spoteo; media = spot 30"; alta = branded content/TP/entrevista)

**Estudio de Inteligencia Social — Clorets**

*Mondelēz México | Reporte individual por marca*

Fecha: 8 de junio de 2026 | Autor: Manus AI

# 1. Resumen ejecutivo

Clorets se analiza como una marca de Goma/menta de frescura / confianza interpersonal / comida mexicana. Marca experta en aliento con oportunidad de apropiarse de comidas intensas, cercanía social y confianza posterior al consumo. El objetivo de este reporte es identificar cómo se expresa la opinión orgánica alrededor de la marca, qué tensiones conversacionales emergen, qué territorios pueden construirse y qué acciones reputacionales conviene priorizar para una siguiente fase de escucha profunda.

El análisis se basa en señales públicas disponibles en redes sociales, videos, publicaciones editoriales y páginas de marca. No sustituye una medición cuantitativa con acceso nativo a plataformas, pero sí ofrece una lectura estratégica inicial de lenguaje social, códigos culturales y oportunidades de inversión reputacional.

# 2. Metodología aplicada

Se aplicó una lectura de inteligencia social enfocada en voz orgánica y comportamiento expresivo de usuarios. La exploración priorizó espacios donde las personas suelen hablar de marcas de manera espontánea: Facebook, Instagram, TikTok, YouTube, Reddit, X y medios digitales que documentan campañas o reacciones sociales.

La lectura transversal muestra que Clorets no debe analizarse solamente como producto, sino como objeto cultural situado. La opinión pública aparece en formas distintas: preguntas, memes, nostalgia, comparación, crítica creativa, solicitudes de receta, reacción a campañas, videos de prueba y comentarios de uso cotidiano. Estas expresiones constituyen materia prima para construir reputación porque revelan qué espera la gente que la marca habilite en su vida diaria.

# 4. Tipos de comentarios esperables

|  |  |
| --- | --- |
| **Tipo de comentario** | **Qué significa para la marca** |
| Comentario afectivo | Expresa amor, recuerdo, infancia, antojo o apego emocional. |
| Comentario funcional | Pregunta por receta, disponibilidad, ubicación, ingrediente, sabor, precio o modo de uso. |
| Comentario crítico | Cuestiona precio, calidad, campaña, ingredientes, autenticidad o pertinencia cultural. |
| Comentario performativo | Usa la marca para hacer humor, meme, reto, reacción, comparación o contenido propio. |
| Comentario identitario | Conecta la marca con mexicanidad, generación, familia, comunidad, fandom o estilo de vida. |

# 5. Tensiones conversacionales

|  |  |
| --- | --- |
| **Tensión** | **Lectura estratégica** |
| Funcionalidad médica vs. humor social | Actizol debe traducirse a situaciones mexicanas sin sonar clínico. |
| Vergüenza vs. confianza | El mal aliento se gestiona mejor desde libertad social que desde humillación. |
| Experto vs. cotidiano | La marca puede ser autoridad relajada en comidas intensas. |
| Nostalgia limitada vs. utilidad presente | La oportunidad principal parece estar en ocasiones actuales, no solo en archivo publicitario. |

Las tensiones no deben leerse como problemas aislados; son zonas de energía social. Una tensión bien gestionada puede producir conversación, identificación y diferenciación. Una tensión ignorada puede convertirse en crítica recurrente o desgaste de confianza.

# 6. Mapa de territorios de conversación

|  |  |
| --- | --- |
| **Territorio** | **Descripción estratégica** |
| Come con todo, habla con confianza | Tacos, cebolla, ajo, trompo, posadas y comida callejera. |
| El aliento social | Ligue, oficina, transporte, fiesta, celebración y cercanía. |
| La Guía Verde | Recomendaciones de comida y rituales con solución de aliento integrada. |
| Experto sin sermón | Credibilidad funcional con tono de complicidad. |

El territorio con mayor potencial inicial para Clorets es “Come con todo, habla con confianza”, porque permite conectar una verdad cultural observada con un rol claro para la marca. Desde allí pueden diseñarse contenidos, activaciones, respuestas comunitarias y métricas de reputación más finas que el alcance bruto.

# 7. Recomendaciones accionables

|  |  |
| --- | --- |
| **Prioridad** | **Recomendación** |
| P1 | Desarrollar un sistema editorial de comidas intensas mexicanas donde Clorets habilite el disfrute completo. |
| P2 | Monitorear términos de ocasión como cebolla, tacos, beso, junta, fiesta, gritar gol y “me huele la boca”. |
| P3 | Evitar comunicación basada en vergüenza directa; privilegiar seguridad, humor y permiso para disfrutar. |
| P4 | Hacer de Actizol un motivo de creencia simple, repetible y culturalmente aterrizado. |

# 8. Riesgo reputacional a monitorear

Riesgo principal: quedar demasiado funcional o clínico, perdiendo relevancia cultural frente a marcas que hablan de frescura con códigos más emocionales.

Se recomienda crear una taxonomía de escucha con alertas por palabras clave, evolución de sentimiento cualitativo, aparición de memes, preguntas frecuentes y comentarios de decepción. Para esta marca, la escucha debe distinguir entre crítica destructiva, fricción útil y humor que en realidad fortalece cercanía cultural.

# 9. Implicaciones para inversión

Clorets justifica inversión reputacional si la marca logra convertir las señales orgánicas detectadas en una plataforma sostenida. La inversión no debería limitarse a pauta o campaña puntual; debe incluir escucha continua, community management con inteligencia cultural, banco de respuestas, pilotos de contenido y medición cualitativa de conversación.

|  |  |
| --- | --- |
| **Dimensión de inversión** | **Recomendación** |
| Escucha social | Monitoreo continuo de temas, memes, preguntas y tensiones específicas de la marca. |
| Contenido | Piezas nativas para los territorios detectados, con formatos diferenciados por plataforma. |
| Comunidad | Respuesta útil o lúdica según el tipo de comentario predominante. |
| Reputación | Narrativa clara que explique por qué la marca importa culturalmente, no solo qué vende. |
